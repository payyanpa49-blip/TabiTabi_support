# TabiTabi プライバシーポリシー

制定日: 2026年5月5日
最終更新日: 2026年5月5日

TabiTabi Apps チーム（以下「当チーム」といいます。）は、当チームが提供する旅行しおりアプリ「TabiTabi」（以下「本アプリ」といいます。）における利用者情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

## 1. 取得する情報

本アプリでは、サービス提供のため、以下の情報を取得または保存することがあります。

### 1.1 アカウント情報

- メールアドレス
- Firebase Authenticationにより発行されるユーザーID
- ニックネーム
- 年齢
- 性別
- メールアドレス確認状態

なお、パスワードはFirebase Authenticationにより管理され、当チームが平文で取得または保存することはありません。

### 1.2 旅行・予定に関する情報

- 旅行タイトル
- 旅行の開始日、終了日
- スケジュールのタイトル
- 日時
- 場所
- カテゴリー
- 予算など、利用者が入力した予定情報

### 1.3 QR情報

- QR情報のタイトル
- 説明文
- URL
- 紐づく旅行、スケジュール情報
- 作成日時、更新日時

現行のMVPでは、QR画像ファイルは端末内に保存され、Cloud Storage等のサーバーにはアップロードしません。

### 1.4 写真情報

利用者が選択した旅行写真、スケジュール写真を本アプリ内で表示するために利用します。

現行のMVPでは、写真ファイルは端末内に保存され、Cloud Storage等のサーバーにはアップロードしません。端末の削除、アプリの削除、機種変更などにより、端末内に保存された写真が失われる場合があります。

### 1.5 位置情報

本アプリでは、旅行中の移動ルート記録、GPS履歴表示、共有用GPSデータ作成のため、利用者の許可に基づき位置情報を利用します。

取得または保存する可能性がある位置情報は以下のとおりです。

- 緯度、経度
- 位置精度
- 高度
- 速度
- 取得日時
- GPS記録セッション情報
- 共有用に作成されたGPSルート情報

バックグラウンドでのGPS記録機能を利用する場合、本アプリはバックグラウンド中にも位置情報を取得することがあります。

## 2. 利用目的

当チームは、取得した情報を以下の目的で利用します。

- 本アプリへの登録、ログイン、本人確認のため
- 旅行データ、スケジュール、QR情報を保存、表示、編集、削除するため
- GPS記録、移動ルート表示、GPS履歴共有機能を提供するため
- 不具合調査、利用者からのお問い合わせ対応のため
- 不正利用の防止、セキュリティ確保のため
- 本アプリの機能改善、品質向上のため

当チームは、取得した情報を広告配信や第三者広告のためのトラッキングには利用しません。

## 3. 第三者サービスの利用

本アプリでは、以下の外部サービスを利用します。

### 3.1 Firebase

本アプリでは、認証、データ保存、セキュリティ管理のため、Google LLCが提供するFirebaseを利用します。

利用する主なFirebase機能は以下のとおりです。

- Firebase Authentication
- Cloud Firestore

Firebaseに保存される情報には、メールアドレス、ユーザーID、プロフィール、旅行情報、スケジュール情報、QRメタデータ、GPS記録、共有用GPSデータが含まれる場合があります。

### 3.2 Apple MapKit / CoreLocation

本アプリでは、地図表示および位置情報取得のため、Appleが提供するMapKitおよびCoreLocationを利用します。

位置情報へのアクセスは、利用者が端末の権限設定で許可した場合にのみ行われます。

## 4. 第三者提供

当チームは、以下の場合を除き、利用者の個人情報を第三者に提供しません。

- 利用者の同意がある場合
- 法令に基づく場合
- 人の生命、身体または財産の保護のために必要がある場合
- 本アプリの提供に必要な範囲で、Firebaseなどの外部サービスに情報を保存または処理させる場合

## 5. 情報の保存期間

当チームは、利用者が本アプリを利用する期間、またはサービス提供に必要な期間、取得した情報を保存します。

利用者がアプリ内で旅行、スケジュール、QR情報、GPS共有情報を削除した場合、当該情報は削除されます。ただし、バックアップ、ログ、外部サービスの仕様等により、一定期間残存する場合があります。

## 6. 利用者による管理

利用者は、本アプリ内で、旅行、スケジュール、QR情報、GPS共有情報を編集または削除できます。

アカウント削除、保存データの削除、登録情報の確認を希望する場合は、「10. お問い合わせ」記載の連絡先までお問い合わせください。本人確認のうえ、合理的な範囲で対応します。

## 7. 位置情報の権限

位置情報の利用は、iOSの設定からいつでも許可、拒否、変更できます。

位置情報の利用を拒否した場合、GPS記録、移動ルート表示、GPS履歴共有など、一部機能が利用できない場合があります。

## 8. 安全管理措置

当チームは、取得した情報について、不正アクセス、紛失、漏えい、改ざん等を防止するため、Firebase Security Rules等を利用し、合理的な安全管理措置を講じます。

ただし、インターネット通信またはクラウドサービスの性質上、情報の安全性を完全に保証するものではありません。

## 9. プライバシーポリシーの変更

当チームは、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、本アプリ内または公開ページ上で通知します。

変更後のプライバシーポリシーは、公開された時点から効力を生じるものとします。
